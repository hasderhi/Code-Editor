HTML Editor

The HTML Editor is a lightweight, user-friendly text editor designed for editing HTML, CSS, JavaScript, and Markdown files. It provides a range of features to enhance productivity, including syntax highlighting, tag auto-completion, and a built-in browser preview. The editor is built using Python's tkinter library, making it cross-platform and easy to use.

Key Features

  1. Syntax Highlighting
    • Supports HTML, CSS, JavaScript, and Markdown syntax.
    • Highlights tags, classes, properties, functions, variables, keywords, strings, and comments.
    • Automatically updates as you type.
  2. Tag and String Auto-Completion
    • Automatically completes HTML tags and strings.
    • Can be toggled on/off in the settings.
  3. File Management
    • Create, open, and save files with support for .html, .css, .js, .md, and .txt formats.
    • Auto-save feature (optional) saves the document every 10 seconds.
  4. Safe Mode
    • Prevents accidental edits by disabling the text area.
    • Useful for viewing or reviewing code.
  5. Find and Replace
    • Search for text and replace it with another string.
    • Highlights all matches in the document.
  6. Zoom and Font Size Adjustment
    • Adjust the font size using a slider or keyboard shortcuts (Ctrl + and Ctrl -).
    • Zoom in/out for better readability.
  7. Browser Preview
    • Open the current document in a web browser with a single click (Ctrl + R).
  8. Appearance Modes
    • Switch between Light, Dark, High Contrast, and Black/White themes.
    • Customizable syntax highlighting colors for each mode.
  9. Templates
    • Choose between some generic website templates with different color themes.
    • You can also insert basic HTML structures (See shortcuts below)
  10. Keyboard Shortcuts
    • Ctrl + S: Save changes.
    • Ctrl + Shift + S: Save as.
    • Ctrl + O: Open a file.
    • Ctrl + N: New file.
    • Ctrl + R: Open in browser.
    • Ctrl + Z and Ctrl + Y: Undo/Redo
    • Ctrl + F: Open the Find and Replace window.
    • Ctrl + U: Open the zoom adjustment window.
    • Ctrl + + or Ctrl + -: Zoom out or in
    • Ctrl + H: Insert basic HTML template

  11. Settings Menu
    • Toggle features like auto-save, tag completion, and safe mode.
    • Access information about the editor and view the license.

How to Use the Editor

  1. Starting the Editor
    • Run the main.py file to launch the editor.
    • The editor opens with a blank document by default.
  2. Creating a New Document
    • Click the New button or press Ctrl + N
    • A new blank document opens.
  3. Opening a File
    • Click the Open button or press Ctrl + O
    • Choose a file to open from the file dialog.
  4. Saving a File
    • Click the Save button or press Ctrl + S
    • Use Save As Ctrl + Shift + S to save the file with a new name or location.
  5. Editing Code
    • Type your code in the text area.
    • Syntax highlighting will automatically apply based on the file type.
    • Use > to auto-complete HTML tags and " to auto-complete strings.
  6. Previewing in a Browser
    • Click the Run button or press Ctrl + R
    • The file will open in the default browser.
  7. Changing Appearance
    • Go to Settings > Appearance to switch between themes.
    • Choose from Light, Dark, High Contrast, or Black/White modes.
  8. Using Find and Replace
    • Press Ctrl + F to open the Find dialog.
    • Enter the text to find and the replacement text (optional).
